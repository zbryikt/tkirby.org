WordPress 使用 Zenphoto 簡易入門 (for sharon)

這邊簡單的介紹一下在 wordpress 裡頭使用 zenphoto 的步驟。

  1. WordPress 發表文章, 這是一定要的 (不然照片要放哪裡呢? ^-^)
    按 Posts, 選擇 Add New

  2. 很好, 編輯窗出來囉~ 接著選則功能列上面的 ZP 按鈕
  3. 按下按鈕後會出來一個對話框, 要你選擇你要用的相簿; 在選單裡選你要的相簿後, 按 “Select”
  4. 按下去之後會跑出一排選單跟照片, 要注意的是你可以在 [+] Image Size 選單中設定你要的照片大小。預設是預覽小圖( thumbnail ), 我想你應該會想選 full size 或是自己給個 size 囉~
  5. 再來就是選要插入的照片, 看小瑩笑得多開心, 就決定是你了！小瑩！
  6. 於是精美的照片就這樣被我們從 zenphoto 插入到 wordpress 囉 🙂 怎麼樣, 方便吧~
  7. WordPress 可以對圖片做一些設定, 你在編輯文章時若點擊圖片, 左上角會出現一個小照片 (Edit Image), 點選他之後, 可以對照片做一些進階的設定, 像是圖片大小 (by percentage), 圖片的樣式表等等的, 可以自已玩玩看囉~

範例：歡樂小瑩
