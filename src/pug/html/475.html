新竹夜八景 / 天公壇、靜心湖、城隍廟與北門大街

前篇：新竹夜八景 / 麗池、迎曦門、火車站

先讓我們複習一下夜八景是哪些：

  • 護城河、東城門週邊景點
  • 城隍廟、北門大街
  • 新天鵝堡及麗池九曲橋
  • 新竹火車站
  • 景觀大道
  • 港南濱海風景區
  • 光復路、介壽路口 LED 光柱
  • 天公壇吊橋

( 資料來源: 新竹夜八景，新竹市政府交通處 )

§天公壇吊橋§

接續上一篇的站前夜八景，接下來我來到天公壇吊橋。天公壇吊橋的位置大約是在這裡：

[mappress mapid=”6″]

看過了美麗的迎曦門及麗池九曲橋，對接下來的五景相當的期待；天公壇吊橋這名字雖然很普通但是難得市內有個吊橋！還是八景之一！一定很不賴吧！那麼很快的我就到了這天公壇公園，就在天公壇的旁邊，這個天公壇是一間廟，原名新闕殿，主祀玉皇大帝也就是天公，所以稱為天公壇了，是清乾隆年間建造的，已經大概兩百多年了，旁邊流著的叫做客雅溪，吊橋即建造在客雅溪上；這一帶叫客雅里 ( 天公壇地址是新竹市中山路431巷36號 )，附近還有一個客雅大道，看起來還不賴的一個地方

天公壇的牌樓金碧輝煌，拍起來格外閃亮



今天重點是夜八景之一的吊橋，所以就不要再在這裡摸了！從天公壇過馬路即是天公壇公園，往公園裡頭不遠處走就看到了吊橋。

第一眼看到這個吊橋覺得實在是蠻遜的，完全毀滅了我對新竹夜八景的幻想，真的就只是一條掛滿 LED 燈的吊橋在一條溪上！讓我想起園區內的一些 LED 燈飾，十分的沒有「夜八景」的感覺，也讓我開始省思到底夜景如何去評量。

如果說弄了幾盞燈就叫做夜景，那麼光復路算不算夜景呢？金山街半夜沒有人的感覺也很不賴吧！其實篤行一路的路燈才是真正美吧！看看這台灣橋粱之美裡介紹的，哪一座橋比天公壇吊橋遜色的？沒辦法，今天不是新竹夜九景天公壇 + 台積電夜景，我只好繼續的來逛這座橋，發揮我以前練拍寶特瓶的精神 – 再單調的東西也有迷人的角度！

首先是吊橋另外一端的步道 – 金光閃閃的，很美不是嗎

走到另一端之後比較寬廣了，構圖也比較多變。來幾張吊橋本身




試著拍出了五種不同色調的畫面，包括綠、藍、白、黃、紅，沒想到這裡顏色變化性也蠻大的。事實上我也是分了兩次來，第二次天空多了些雲彩，畫面比較戲劇化，比較好拍



拍完了照，我好像也稍稍的抓到了這裡的感覺。

§城隍廟、北門大街§

離開天公壇吊橋後我前往新竹城隍廟，雖然不是新竹在地人不過這幾年我也算是常在火車站前出沒，可是怎麼好像從來沒看到過城隍廟呢？新竹城隍廟的位址：

[mappress mapid=”7″]

如果是這個位置的話我好像常經過，有時候還會有抓違規逆向行駛的警察出沒。到附近之後我決定四處繞繞，先繞進了一個市場；這是一個有點像商店街的市場，一條長長的通道頂上加蓋遮雨，兩旁都是店面，不過也許是晚上了吧，這個市場雖然看起來攤位很多，但是沒有人在逛，店家也只有幾間有開著。我走到後方後發現通往城隍廟的小路，就轉了進去：

結果旁邊是一間賣情趣用品的店，這跟城隍廟實在很不搭阿。看上圖還莫約看得出店名。往前走即通到一個很多攤販的空間，有點像是台北士林集中臨時市場那樣，繞了一圈發現這裡大概是城隍廟前廣場，只是築漸的四周圍了起來，天空也加蓋遮蔽物使得整個成為了一個內部的空間。

角落有一個門上頭寫著慈門，旁邊就是城隍廟的正廳跟前庭了。

走到底轉個彎有個廁所，感覺得出來已經身在城隍廟中了，不過四面都圍了起來，天空也多少遮蓋住了。走回攤販區從旁邊的門走出去，就回到了熟悉的新竹市區馬路。整個城隍廟外圍被攤販所包圍，感覺大約就像是一間廟被罩在另一個建築物中間，然後四周都是攤販。難怪我不知道城隍廟在哪了。

當然外面還是有牌樓跟招牌指示城隍廟，不過蚵仔煎的招牌也不遑多讓的搶眼呢。



由北走出走出城隍廟即是北門街，這一帶原本有個北門不過現在只剩下北門街及遺址附近的人行道上有說明牌了。總之就逛逛吧！




從原路走回去我在商店街旁看到一條小路，便繞了進去看看，原來是一個傳統市場，傳統市場晚上沒有半個人，不過有很多隻貓。讓我想到古老的 RPG 中夜晚在街上出沒的也就是小貓兩三隻；這邊的貓大多怕人，我繞了一圈看幾隻小貓急急忙忙的跑到角落去躲起來看我，卻有一隻白花貓直直的往我走來

原來是隻不怕人的小花貓，在我身邊繞來繞去的還喵喵叫，怎麼啦，你想要吃罐頭嗎？

這時旁邊出現了一隻馬爾濟斯，我想起以前看過網友「神貓村奇遇」的網誌文章，那麼我是不是也來個咒文「去去狗狗走」，就可以討好這邊的貓兒們讓他們全聚到我這邊讓我拍個夠呢，嘻嘻嘻… 正當我在幻想時這隻小花貓就追著逃跑的馬爾濟斯消失了…. 對不起… 原來你是貓中的佛地魔…

旁邊有另一隻黃虎斑貓，雖然也不怕人，不過也不親人。說不親也不是不親，因為他還是會過來，只是都不太看我。這要叫做傲嬌還是口嫌體正直呢？

這時佛地魔回來了。他還用頭頂我的屁股！


小貓看著我離開的表情實在是讓人捨不得走，可是還是得回家的阿！

之後還去了靜心湖，雖然不在新竹夜八景之中，但我認為這個地方可以媲美麗池了。有機會再完整的連同之後兩景一起發表出來。剩下來聽說景觀大道的夜景還不錯，不過南寮好遠阿，而且晚上安不安全阿？看來下篇夜八景系列文遙遙無期了….

靜心湖一角



話說回來，還有一景呢？

§光復路、介壽路口 LED 光柱§

時常都會經過介壽路、光復路口，但我沒想到這個東西竟然可以成為夜八景之一，這比起天公壇吊橋來說，更值得納入新竹新‧夜七大不可思議之中。

基本上這個 LED 光柱指的應該是介壽路、光復路口，以及園區一路、光復路口的道路分隔島上的一排光柱。我實在不知道這有什麼好夜八景的，園區內有一顆 LED 櫻花樹要不要算夜八景？總之我連寶特瓶精神都沒有發揮，因為在道路中間，原本想說趁半夜去晃一下，結果拖到現在整個就是要被我跳過了。

為了避免大家白跑一趟，地圖我就不貼了 XD
