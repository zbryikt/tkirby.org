老英格蘭莊園

週末跑到了清境的老英格蘭莊園渡假。

先前對這間旅館的印象，只有上合歡山爬山時路過時的讚歎還有前陣子新聞報導上看到的違建風波。最近剛好有個機會要找個地方好好玩個兩天，不知怎麼的忽然就想起這個地方；每次都是順道路過看看風景，這次不如來住住看吧！

老英格蘭最主要的特色就在於其都鐸式風格的建築，每當驅車路過時他高聳的鐘塔總是相當的搶眼。我們從台北直接開車到這裡，遠遠的就看到那個鐘塔，完全不用擔心開過頭。話說，如果在 Google Maps 上輸入「老英格蘭莊園」的話，他會指到另一個地方，不確定是不是相關的民宿，這次所指的老英格蘭莊園的位置是在 Google Maps 上「老英格蘭精品旅店」的位置。

老英格蘭莊園就在台14甲線公路邊，三棟組成的建築群，A 棟(最靠馬路的建物)為櫃台所在，中間的 B 棟則為鐘塔該棟，同時餐廳及咖啡廳也在這裡。C 棟則有一個宴會廳、音樂廳。 A 棟及 B 棟是相通的， 而要到 C 棟則要繞出庭院。庭院與停車場以欄杆隔開，之間算是一條通道，因為在莊園的後方還有其它的民宿房舍。現場很多遊客在觀光，但只有住客可以開車進去停，房舍跟庭院也只有房客可以進去，在這裡的遊客多是以建物為背景拍照，或是欣賞往山下望去的雲海。

另外老英格蘭莊園附有下午茶咖啡廳(庭園露天吧 / Outdoor Baclony)，另外在鐘樓的頂端有個 Henry Bar，不過目前都沒有提供餐飲服務，聽服務人員提到好像是冬天因為天氣的原因所以沒有開放，所以我們沒有參觀到鐘樓頂端的 Henry Bar 。

雖然外圍很多遊客，但進入旅館範圍內後，只有住宿的房客可以進入，而旅館本身的房客似乎沒有愛在院子裡晃，基本上沒有什麼人；庭院本身很漂亮很適合拍照，如果喜歡拍照的話，我想在這裡頭晃一整個下午大概都沒問題吧。




我們從我們房間所在的 C 棟晃到了 A 棟，上到 A 棟的二樓的戶外庭院，再到了二樓的陽台。建物內的公共空間多可以自由進出，除了戶外隨處有庭院，室內也有許多可以駐足的點，整個莊園就像個小迷宮，每個角落都有用心設計過的感覺；有噴水池、拱門、有涼亭、似乎也有游泳池跟 SPA 池(冬天未開放)。我們很單純的就走到哪玩到哪，莊園裡儼然成為了清境這一帶的景點。

回到我們住的房間，我們住的是「Fantasia Suite / 幻想曲二人套房」，其實由於訂房時只剩這間了，也沒得選；這間房的位置在 C 棟左方的頂樓，視野不錯，房間旁邊的樹恰逢寒流來都紅了。

室內空間相當寬敞，兩張床是 Queen Size 的，雖是兩人的房型，睡四人也是綽綽有餘。每張床上都有三種尺寸的枕頭各兩個！

室內的設施有壁爐形式的電暖爐、百萬等級的音響、液晶電視；在門口旁附有流理台，除了一般生水外也附有可直接飲用的水龍頭。另外，紅酒、餅乾、水果、冰箱 Mini Bar、茶包等較高級旅館會提供的東西這裡都有，並且是不需另外計費的。

莊園在櫃台處還提供了相當數量 CD 出借的服務，有發燒片、古典樂、也有流行歌集，在 Check-in 時服務人員即會提醒你可以借個幾片帶回房間放。我選了幾張古典樂集，包括舒曼、拉赫曼尼諾夫、拉威爾跟德布西，不過待在房間裡的時間並不算長，只聽了其中三片。

窗外看出去的景緻也是相當的不錯：

check-in 的時間是下午三點半，我們的晚餐在五點半開始供應，大約在太陽下山後我們就走到餐廳去用餐。餐廳上方應該是露天咖啡座，因為吊燈的上方是透明的玻璃。晚餐是採自助式的 Buffet ，有冷食及熱食、湯品、數種飲品及 Movenpick 的冰淇淋。進來用餐只要跟服務人員報房號他便會為你帶位。餐點的內容基本上我們都滿意，這是頓相當享受的晚餐。雪倫一登場就把兩朵燻鮭魚花給夾走了，真地是狠角色！

餐廳本身的裝潢也是相當的優。

吃完晚餐大約也七點多了，走出餐廳迎面而來的是 … 老英格蘭莊園的夜景。於是我們只好繼續拍照。晚上的老英格蘭外圍的遊客較少，但景色不比白天遜色，甚至更多了些變化性。

在 C 棟宴會廳旁的長廊兩側擺了許多古代偉人的頭像，我們到時剛好燈點著了。隔壁的莎士比亞廳裡正有人在拍婚紗，而晚上這裡將舉行一場小型音樂會。順道一提，我詢問工作人員關於婚紗的事項，若非住客要在這裡拍攝婚紗，總共的費用約在 11000 上下。雖然這裡拍婚紗超正點的，不過若是預算不足，正在準備婚禮的男士們可能要小心點別讓未婚妻想到要來這邊拍婚紗囉。

在晚上的音樂會開始之前，我們跑到莎士比亞廳坐著休息等著時間到。我還是拿著相機在裡頭走走拍拍，這裡面的裝潢也是相當華麗，這裡也有一個吧台，同時莊園中千萬級的音響即放在這裡，還擺了一台河合的平台鋼琴。我試彈了兩首，感覺相當不賴。坐在裡面真的很有超現實的感覺。同時間我也用移軸鏡拍了幾張。

晚上的音樂會基本上就是透過千萬級的音響欣賞 CD ，同時請來了一位小號手，除了以 CD 做伴奏外同時與主持現場的龐克頭工作人員鋼琴二重奏，印象中演奏了花之二重唱、Saints go marching, 夜來香, Take Five, Danny Boy 等等的曲子。

音樂會結束後，我們也回房休息了。隔天的退房時間是早上 11:30 ，早餐供應則在 07:00 至 10:00 。基本上我們回去休息之後，隔天大約就不會太有時間在這裡逛了，因為我們通常會睡蠻晚的，吃完早餐就要準備退房了。早餐還不錯，不過沒有雪倫最愛的現煎 omelet ，相信這是一個可改進的點 XD。

那麼，我們拿著意見回函與房門鑰匙到櫃台換取了神秘小禮物後就回家囉，還真的有點捨不得走呢。真的很不錯的地方！將來有機會的話會再訪的！
