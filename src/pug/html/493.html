N3DS – 任天堂 3DS / Aqua Blue 開箱

二月底任天堂推出了新世代的掌上型遊戲機 N3DS，所以我今天就跑去台北地下街買了一台！



3DS 最大的特點就在於他的上螢幕顯示的是可以直接裸眼觀察的 3D 立體影像，而也因為要用眼睛看，所以沒有人能在平面媒體上展示其效果，只有透過實際操作才能感受他的炫麗阿！看一下盒子裡面應該要有什麼東西：

打開盒子把主機拿出來，跟卡比比較一下大小

我買的是藍色的主機，外殼其實相當有金屬質感，而且有層次的感覺，讓我用能量消波塊來測試一下

我買的版本是隱藏限量卡比版，從某個角度觀察外殼會看到淡淡的卡比反射 (大誤)

外殼也是 3D 的！？

這次主機的設計將電源鍵直接做在下螢幕旁，而且在下螢幕左邊增加了一個類比式的搖桿

長按 Power 鍵會出現詢問「休眠或是關機」的對話窗。新的搖桿使用起來就好像是在推動一枚硬幣一樣：

原本 NDSL 在右下角的 Start / Select 鍵則是被放到下螢幕的下方，連帶著一個新的按鍵「HOME」。Home 按鍵的功能類似 Wii 搖桿上的 Home 鈕，可以讓玩家在任何時候直接跳回到開機後的主畫面。耳機孔則是做在 Home 鍵的正下方了。

上螢幕右方多了一個 3D 的開關，可以調節 3D 的程度避免玩太久頭昏

當畫面可以產生 3D 效果時，綠色的 3D 燈會亮起以指示玩家：

N3DS 上也有 Mii 頻道：

右上角有電池電量顯示 ( 不過這個電池並不是立體的 ) ：

上方也多了一個攝影機，用來拍攝玩家的照片：

來看一下主機的四周。背後，據說使用盜版卡會在主機上留下紀錄：

左方，有音量控制鍵與 SD 卡插槽：

出廠時就已內附了一張 2G 的 SD 記憶卡：

充電器插槽與右按鍵：

正前方有耳機孔、電源指示燈與充電指示燈

上蓋下方有兩個攝影機，可以讓主機拍出立體的影像，可以用在擴增實境 ( Argument Reality, AR ) 的效果上。買主機即會附六張所謂的「AR」卡，將之置放在桌上以主機拍攝後，會在上螢幕呈現出立體的世界並出現虛擬的角色。

看！科比出現了！(純示意圖)

連林克也一起出來湊熱鬧了！超立體的，真是厲害！

由於首發遊戲一時間也不知道買哪一個，最後我買了任天狗，還附帶敗了一片太鼓之妖怪大決戰！

3DS 遊戲的內盒與 NDS 的內盒稍微有點不一樣，不知道是為了節省成本還是設計，透過盒子的洞，封面內頁也被妥善的利用了：

3DS 的遊戲卡比 NDS 的多突出了一塊，我猜也許是為了防呆避免 3DS 的遊戲被插入 NDS 內。這次除了狗以外好像還有任天貓！

妖怪大決戰如同之前的版本有附兩隻太鼓版觸控筆：

NDS 的遊戲片沒有突一塊出來

看我用 3DS 來玩 NDS 太鼓！

松難度初見全接是一定要的！可惜良率太低！

還有神秘的 RPG 模式，戰鬥是用演奏模式解決

3DS 還有附充電座，需要充電的話把主機往上一擺就行了，不需要再另外做插電的動作，相當方便！不過缺點就是稍微佔空間：

很期待可以玩到 3D 的瑪莉歐或是薩爾達傳說！還沒買的人趕快去買吧！
