g0v.tw hackath0n / 零時政府第零次動員戡亂黑客松

在充滿旅遊攝影跟開箱文的部落格裡忽然出現一篇黑客松的文章真的蠻突兀的，不過事實就是這樣，我其實是個軟體工程師，在科技業裡載浮載沈。原本想要偽裝成活動攝影師然後不寫 code，可是野心被 clkao 識破所以還是生了一個 project 叫做「公眾人物關係圖」。雖然邊拍照邊寫 code 效率比較差一點，不過原本就想幫活動作個記錄，順便寫篇網誌，最後 project 也是有把概念做出來，應該算是值得了。於是就出現這一篇文章囉，最後決定照這個網誌的風格以照片為主來寫活動記錄 (其實是偷懶想少寫些字..)。

Anyway，此活動緣起自我們參加 Yahoo Open Hack 2012 的得獎作品，再追根究底則是起因於行政院的「行政院經濟動能推升方案」這支廣告。當時 Opop 與 CA 找我跟 clkao 一起參加 Yahoo Open Hack 2012 ， clkao 看到了這支據說花了 119 萬的行政院廣告，認為國家實在是愛亂花錢，於是就開始着手把行政院主計處提供的政府年度總預算資料丟到開放平台上，同時我們並決定以中央政府總預算視覺化為參賽題目。在 Yahoo Open Hack 網頁還看得到我們的作品列在其中，名為「 TW Budget 」，最後獲得佳作並得到五萬元獎金。

Yahoo Open Hack Taiwan 2012 的盛況：

激情過後，我們仍繼續開發該網站並改用 g0v.tw 作為網址為 gov 的偽裝版，開始有了零時政府的概念，延伸網站觸角的想法也漸漸成型，於是我們決定把獎金拿出來辦 hackathon ( ~= programming marathon ) 活動，透過網路熱心人士的力量，讓事情真的就像 119 萬廣告中所說的，「很多事情都在加速進行中」。

整個活動基本上都是 clkao 與 opop 辦起來的， opop 負責食物，其他的則由 clkao 一手包辦。為了出點力我也幫活動拍了些照片，於是就催生了這篇文章囉。活動經由 g0v.tw facebook 專頁 及 IRC 宣傳，透過 registrano 報名，在活動前大家先在 IRC 頻道 #g0v.tw@freenode 討論以及透過 Hackpad 交流資訊，當天報到時對該日大約會進行的主題還有可能的成員大致上都有一些概念，於是在每個人簡短的自我+主題介紹之後(大約也花了一個多小時吧)， Hackathon 便正式開始!

還有現場轉播：


其實最初 clkao 原本是打算在咖啡廳舉辦活動的, 結果這個小型 Hackathon 因為報名踴躍不斷地加開名額而決定更改場地至中研院資創所(感謝資創所同仁的協助)，最後約有六十人報名吧，扣掉少數沒有出現的現場仍有相當多人，看來對政府花大錢作無用廣告不滿的人相當的多啊。

活動開場在資創所地下室的會議室，由於人數多，活動也借了一樓的另一間會議室來使用，Hackathon 開跑之後大家尋找志同道合的夥伴就地尋找位置開始進行 coding 。看這裡已經圍成一圈在準備進行了：

熱烈討論中：




午餐是披薩，沙拉，炸雞，還有西米露甜湯跟可樂。反正獎金是賺到的，就儘管拿來用吧，只是要是吃不完那就要外帶了，幸好 Opop 量抓得還蠻剛好的。


活動進行到約下午5點接近尾聲，由有做完一定程度的各組分別進行成果分享；有不少組做出了一些初步的結果，包括台灣各鄉鎮市區人口密度的視覺化，childnotfound.tw，立法院公報分析等等，因為當時忙著攝影有些主題沒聽清楚，clkao 在網誌中有比較詳細的說明。
