Nikon D3100 + Nikkor 35mm f/1.8G 開箱

買了一臺 Nikon D3100 單眼相機給雪倫當生日禮物，搭配兩顆鏡頭：

  • Nikon 18-55mm f/3.5~5.6 VR? (Kit)
  • Nikon AF-S DX 35mm f/1.8G

雪倫一直都是使用我借他的 Nikon D80。還記得我剛從一般小感光器的數位相機換到 Nikon D80 時，對他拍出的照片時感到相當的驚豔，不管是那細膩的畫質或是淺景深對只用過消費型數位相機的我都是完全沒有接觸過的；當然也因為滿機身的按鈕而手忙腳亂，不是拍出全黑就是拍出全白的照片。

後來漸漸的玩了各式各樣的鏡頭及機身，對照片畫質也越來越要求了，也發現 D80 有一些地方沒有辦法滿足我，比方說可用 ISO 並不高，照片容易偏紅等等的問題，也因此換了機身，而空下來的 D80 就給雪倫玩囉 – 當然他也就跟著遇到了同樣的困境 – 在室內常拍出紅通通、黑漆漆又糊糊的照片。當然這些問題透過調整相機設定或是更換鏡頭可以得到一些改善，但這就使得操作機身多了些難度。

說實話， D80 是 2006 年的產品，我大約是在 2008 年夏天買的，而現在已經 2011 年了，算是五年前的機型；這些日子 Nikon 陸續的推出了許多的新機，比方說 D90、D3000、D5000、D7000 等等。規格也是不斷的在改變，從感光器清理到錄影功能；我想也許是時候將 D80 淘汰掉改換新機身的時候了，正好雪倫的生日也到了，就買一台相機當成禮物送給他，順便連開箱文都幫他寫囉！(其實是他叫我寫的)

由於這是要給他用的相機，考量點自然就以他的需求為主，就是要機身輕巧、照片明亮。從這個角度出發很自然的就會想到 EVIL 機，因此我比較了數個品牌的 EVIL 機包括 Panasonic GF2 / LX5、Sony NEX5、Samsung NX100、Olympus EP2 以及 Nikon D3100 入門機。我從幾個面向去考量，包括價錢、可用 ISO、重量及體積。

  • 價錢的範圍大致上從 14000 到 30000，基本上都沒有突破我的預算。
  • 可用ISO 一般都在 800 左右，只有 NEX5 跟 D3100 有達到 3200 。 (這點比較主觀，我是參考一些評比網站及試拍照分析的)
  • 重量都介於 200g 到500g 之間； NEX5是 350g 左右(加 16mm 餅乾鏡)，D3100 則是650g(加35mm 定焦鏡)
  • 體積則是介於 110 x 63 x 34 ~ 132 x 96 x 77 之間 (長 x 高 x 寬)。


若要高 ISO 可用那就是選擇 NEX 或 D3100，實際上到店家試玩機器的心得則是覺得D3100 較容易操作。再加上 NEX 目前鏡頭群不多，而我已經有為數不少的 Nikon 鏡頭了，於是就選擇了 D3100。至於鏡頭的話，50mm f/1.8D 雖然非常小巧但焦段對 APSC 相機太大， 35mm f/2 又沒有鏡頭馬達，24mm f/1.4 太貴，Sigma 廣角三兄弟太大太重，於是就選擇 35mm f/1.8G 做搭配。D3100搭配Kit 鏡與單機身間的價差只有一千多，就順便把18-55mm Kit 鏡帶回來了，是顆相當輕的鏡頭！

那麼就先來打開 Nikon AF-S DX 35mm f/1.8G 這顆鏡頭的箱子吧！

照片上可以看出來機身左右都短了一截，鏡頭也比較瘦 (右方鏡頭為 Sigma 30mm f/1.4)。整臺相機可以放在手掌心上，相對於表面上的大小，重量給人很輕的感覺，好像玩具一般。

他比起專業機少了許多東西，比方說機身馬達、前轉盤、機頂 LCD、ISO / WB 按鈕等等。下圖中右下有一個光圈的圖案；意思是若要調整光圈大小要按住這個按鈕然後轉後轉盤。在 M 模式，由於曝光補償在手動模式只有參考的用途，所以就利用”壓住曝光補償鈕”這個做法來做到調整光圈。至於在 A 模式，轉盤調整的是光圈；在S模式轉盤調整的則是快門。拿掉前轉盤也許是為了輕巧單純吧，實際上操作起來的感覺並不會不便，很容易就可以適應了的。(轉到嬰兒情境模式閃燈會跳起，是說小朋友都很會亂動嗎？)

我覺得比較大的問題應該是沒有機頂 LCD；沒有機頂 LCD，只好把資訊直接顯示到 機背的 LCD 上。這代表每次對焦 LCD 都必須亮起來，否則有些資訊沒有辦法立即看到。

相機資訊顯示後可以按機頂的 info 鈕將之關閉，這大概會變成這臺相機的操作習慣之一吧( 對焦完按一下 info )。調整 ISO / WB 的鈕也沒有了。不過相機左前方有 Fn 鈕，預設是設定成調整 ISO? 。其它設定像是 WB、 Quality 也可以在按機背左下角的 <i> 鈕後透過十字鍵選擇調整，是還不需要到按 menu 進選單選取的地步，所以其實並不會太麻煩。

沒有機身對焦馬達這件事使得一些鏡頭沒有辦法使用，原本因為大小考量想選購 Nikon 35mm f/2D 這顆鏡頭的，也因此而換成了稍稍大一點的 Nikon 35mm f/1.8G。

買回來後我及雪倫試拍的結果都還蠻滿意的，雖然小小一台，但反而變得很輕很好拿，按鈕也蠻容易就可以掌握，對於小手的人相當的友善。掛在脖子上脖子也一點 都不會酸。另一顆鏡頭 18-55mm 更輕 (雖然官網 spec 似乎不是這樣寫)，所以裝起來要帶去登山既不佔空間，也完全不會是重裝的負擔。 (還有一點就是，因為比較便宜所以撞來撞去比較不會心疼)

雖然這樣是從中階機跳回到入門機種，不過我覺得就需求面上來講這樣的組合蠻符合我們的要求的。我想畢竟對人來說這仍是工具，與其追求高畫質、高便利性不如認真的思考自己需要的東西到底是什麼，再去做選擇會比較有意思。
