新竹夜八景 / 麗池、迎曦門、火車站

在新竹工作了幾年了偶而會看到路旁公車亭貼著新竹新八景、海八景、夜八景的照片；什麼夜八景阿？最初也沒有多去注意，只是這陣子比較有空檔了，想想新竹都沒有觀光觀光，於是我就想趁著晚上來個夜八景巡禮了。那麼夜八景是什麼呢？根據新竹市政府交通處所提供的連結及網站，是指這些地方：

  • 護城河、東城門週邊景點
  • 城隍廟、北門大街
  • 新天鵝堡及麗池九曲橋
  • 新竹火車站
  • 景觀大道
  • 港南濱海風景區
  • 光復路、介壽路口 LED 光柱
  • 天公壇吊橋

( 資料來源: 新竹夜八景，新竹市政府交通處 )

唷呵，好像還有模有樣的。我是知道有些不賴的景點像是靜心湖、清大交大，倒是上述其中有幾個點我好像常經過，沒注意到過有什麼特別的景色阿？

§麗池九曲橋§

總之我先來到了麗池九曲橋，他在新竹市立動物園的旁邊，食品路上

[mappress mapid=”4″]

基本上我是第一次來到這個地方，晚上看得不是相當清楚其實對整個地形不是相當了解；不過一到這裡馬上就可以看到麗池及對面舊眷村的燈火

麗池1916年建成，對岸的日式建築人稱湖畔料亭建於1932年，是當初日本人接待權貴人士的地方，1950年後後改建用以安置空軍單身飛行員而稱羅漢堂，這些軍人又在此逐漸的成家而形成眷村後更名為空軍十一村，直至現在因眷村改建，住戶遷出，成為目前主要做觀光用途的新竹公園，甚至還有偶像劇來到這裡取景。

九曲橋則是建在麗池上的橋囉。順道一提，九曲橋英文寫做 zigzag bridge，從湖畔往西走馬上就看到了這座橋：

§東城門、護城河、新竹火車站§

新竹市的東城門「迎曦門」離火車站相當的近，所以這兩個景點我就一併參觀了。

[mappress mapid=”5″]

其實來之前我也不知道他叫迎曦門，第一次拍之前我也根本還不知道什麼夜八景的，後來才連帶護城河跟火車站一起把他拍下去。網路上查東城門可以看到很有名的一個構圖是從高樓上往下長曝圓形車軌，不過我並沒有這樣拍，我只是純粹從站前地下道走過去到城門旁晃了晃。

這是新竹竹塹城僅存的城門，竹塹城又是什麼呢？竹塹(音”欠”)是新竹舊的地名，是由荷蘭人時代以前居住於此的平埔族口語音譯而來，現今的新竹城市風貌大致上是在日據時代定下來的。身為二級古蹟的迎曦門落成於西元 1829 年，以花崗岩為主體的兩層式建築，上層原為木造，現已改為混凝土造。

接下來是護城河；這邊我沒有拍很多照片，不過我發現這邊不僅有夜鷺，水面上還有其它的生物：

想喝水喝不到跑好遠才喝到水的貓

這條護城河味道並不會很糟，時常還可以看到旁邊的小廣場在辦小型演唱會等活動。旁邊就是誠品了，文藝氣息頗濃厚，不過坐在河邊的通常都是情侶不是文藝青年。

護城河走到底火車站也差不多到了，所以我就順便找了個地方拍了一張鳥瞰火車站的照片。
