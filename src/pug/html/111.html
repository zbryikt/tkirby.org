Nikon D3s + Nikkor 14-24mm 1:2.8G ED 開箱

我家的狗前幾天從門口叼回了裝著一個金色盒子的塑膠袋，打開一看，不得了了，是 Nikon D3s！

跑去買了一台 Nikon D3s，附帶一顆 Nikon 14-24mm f/2.8G ED，大三元之廣角變焦恆定光圈。

從最早的 Nikon D80 搭配 18-135 kit，到 Nikon D300s開始搭配各式各樣的鏡頭，到今天玩單眼也已經兩年了。前陣子同事買了一台 Canon 1D4，那對焦速度之快和準，看得我不禁口水直流… 結果… 結果就出現這篇文章了。

對焦模式選擇

垂直把手上的快門釋放鈕，可以上鎖；第一次試拍時一直不小心按到，還以為對焦系統有問題

雙 CF TYPE-I 插槽，開啟方式不同於D300s，需要推蓋後按鈕扳開，基本上很難誤開到

屈光度調節器及測光模式選擇，測光模式需按下中間按鈕後撥動轉盤。設在無內建閃燈的機頂，可以看出機頂比 D300s 大很多

釋放模式撥盤，閃燈模式、包圍、鎖定按鈕(鎖定快門 / 光圈)。除了按鈕不同，操縱感是同於 D300s 的。

垂直把手液晶螢幕，可以看到 ISO、Quality、白平衡的資訊，其實也可以顯示其它東西。右邊有錄音鈕，可以替照片錄製註解。

來張拍不清楚的機頂 LCD

Nikon D3s 操作感很接近 Nikon D300s，不過有幾點大家都知道的不同：

  1. 多了一體成型式的垂直把手。
  2. 無內建閃光燈。
  3. 左上方的三顆按鈕在 D300s 原本是 ISO / WB / QUAL，在 D3s 則移到垂直把手上
  4. 在垂直把手處並多了一個液晶螢幕，以顯示 ISO / QUAL / WB 資訊。
  5. 觀景窗內的測光表移到了右方。
  6. 觀景窗變大了！打開記憶卡插槽的方式也有所不同。
  7. 照片縮放是用撥轉盤的方式。一開始很不習慣，現在反而不習慣舊的方式。不過這方式有一個缺點：單手做不到。
