g0v hackath4n / 零時政府第四次國民大會黑客松

g0v 零時政府在兩個禮拜前舉辦了名為國民大會的黑客松。

自去年十二月起舉辦的零時政府黑客松至今已舉辦了五次的活動，催生了許多有趣的專案，包括 404 失蹤兒少協尋、標案分析、立院透明、人物關係、法律亦毒氣(即易讀器)、福利請聽、萌典、路見不平、求職小幫手等等。其中有許多有相當正面的意義，比方說福利請聽透過親切簡單的介面協助身障朋友更容易了解自己應享有的福利，求職小幫手讓所有的求職者很容易就能知道公司是不是有潛在的問題等等。

g0v 本身的口號很容易讓大家誤解：寫程式改造社會、鍵盤革命、阿宅救國 – 曾聽過有人覺得我們是反政府組織，也有人指我們是在野黨黨工，更聽說我們是一群沒素質的工程師。我想也許是因為”改造”, “揭露” 等等的字眼讓大家感到不安，而略帶詼諧諷刺的文化讓大家覺得低俗；但事實上，我們的立場很單純：「我們有所不滿，而我們不怕弄髒手」。也許你會覺得對政府的不滿只能是激進的，但這不盡然為真；舉個例子，這一次的專案「PetNeed.me」目的在於讓各動物收容所的流浪動物更容易曝光，進而提高他們生存的機會，這是一個早一天完成也許就能多拯救一條生命的專案，發起人 jslee 因為這陣子狂犬病的疫情蔓延擔心會讓這些生命更快地被撲殺，因而在這次的活動將原本進行到一半的專案開放出來，希望能夠找到同伴更快地把網站完成。很單純的一個初衷，比起純粹在電腦前看看花邊新聞，罵個兩聲幹就關電視睡覺有意義多了。

( 你會不會覺得「petneed.me」 沒有意義呢？我可以想像一類聲音：「這能救多少條貓狗？」「沒用啦，領養了還不是會再棄養。」「浪費時間。」我也能想像有這樣想法的他們接下來會做的事：「關電視睡覺」。你就是他們嗎？)

若想要再政治一點，你也可以看看政誌，瞭解一下我們政府在過去做過些什麼事。是的，這也是 g0v 近期的一個專案。

政府的本質在於人民為管理自身的環境而組織的機關，換言之，本質即為人民。當我們對政府有所不滿，我們想要付出心力來改善我們的環境，這時的我們即是你，都是台灣的人民。我認為這一個價值在零時政府之中是很重要的；我們對於政府提供的服務不滿意，因此我們捲起袖子來自幹；你們對我們土炮出來的成果不滿意，你自然也可以捲起袖子來跳進這個坑中。當政府這個堤防破了個洞，少數人伸出了手去堵住洞時，在堤內的你最不該的就是以為，嘲笑他們螳臂擋車的你，可以置身事外。

說到堤防， yllan 在這次的黑客松建立了內插式的三維化雨量資訊，讓原本不容易立體呈現的雨量資訊很清楚地呈現了出來。雨量只是個範例，其實很多東西都可以利用這個工具，比方說輻射量、地震震度，紫外線，用電量等等。像這類的貢獻並沒有為社會帶來直接的衝擊，然而他提供了一個簡便的方式更清楚地呈現出資料，未來若有其他人需要向群眾正確清楚地呈現關鍵性的資料，這便會是一個很有用的幫助。資料視覺化、通俗化讓訊息更容易深入群眾、函式庫開發減少工具開發的時程、甚至連只是個概念性的設計我認為都具有其意義。在組裝電腦前，把螺絲起子準備好吧，若少了一把起子，在責怪別人之前，記得先找找自己的口袋。

也許你會覺得你連螺絲起子都生不出來，但任何人都可以用任何方式對社會作出貢獻。你可以做志工，可以掃街，可以常打警廣；即便是安裝一個手機程式，都可能可以幫點小忙。雖然尚在開發中，路平專案提供全台灣道路施工的查詢界面，除了以地圖形式呈現，也包含了道路修繕與交通流量、地方預算及標案狀況的分析以揪出黑心的包商，同時也提供使用者容易回報路況的機制，在這一次個黑客松三位參與者更展示了以手機監測震動狀況的工具，希望可以做到當在特定路段許多用路人都有監測到震動頻繁的情形時，便能自動建立路面可能不平的記錄。

也許你這時又會有些意見了，在說話之前，看看他們的待作項目表(下右圖)，有意見就丟進去吧，沒人做就你來做吧。有位從第零次黑客松開始參加的朋友，為了想做些什麼事，從原本完全不會 sass, jquery 等網路技術，經過很努力的學習之後的現在，其實還是不太會，但他對 404 失蹤兒童協尋、動民主等專案作出了相當大的貢獻。

同樣也是群眾協作概念，這次黑客松 ronnywang 也做了一個新的服務 – 新聞小幫手，這是一個 chrome 瀏覽器中的擴充功能並附帶一個資訊交換平台，讓大家在瀏覽新聞時可以回報問題新聞，這樣在其他人要閱讀新聞時，就可以看到「新聞可能有誤」的提示訊息。我曾經想過類似的專案「URI Comment」，透過加裝瀏覽器延伸功能讓使用者可以對任何一個網頁的任何一段文字產生連結並注解、討論，目的在於更正錯誤的內容以及透過群眾力量激發新的想像空間。新聞小幫手可以說是某種角度及程度上實現了這個想法。


( 此套件藉由分析新聞內容實現提示功能，並不會針對特定網站，上圖僅為其中一個範例。 )

g0v 本身為了減少活動及專案之間協調需要花費的時間跟精神，也做了許多的基礎建設，我認為其中很多的方法都相當適合作為其他活動團體運作模式的樣板的。這樣說吧，g0v 的立意在於透過群眾的力量及開源的方式來改造政府，怎樣最有效率地進行這項任務？我們利用 ethercalc 、 hackpad 跟 google docs 建立了 hackfolder 系統，將文章跟常用的功能集散在同一個地方，你可以看到我們應用在 g0v 上的實例，hack.g0v.tw。連上 hack.g0v.tw，在報名活動後，你可以在 Idea Pool 裡跟其他參與者線上非同步的討論關心的議題，並寫下你的意見；若你的想法已經發酵，你也可以利用空白模板建立一個你想發起專案的頁面，並在專案登記處簽名。同時你也可以透過 People Hub 來尋找潛在的合作夥伴、甚至透過線上聊天室 ( IRC ) 與其他參與者即時交談。這一切都是即時發生在網路上，我們不需要密集的開會，因為討論是透過共筆的方式進行；我們沒有通勤的時間，因為你的電腦即是你的工作室；沒必要面對面會談，線上聊天室即是虛擬的咖啡廳。歡迎你來到二十一世紀。

這一次黑客松也開了其他幾個同樣有意思的專案，例如64885公益律師呼叫系統、立院看門狗、零時通關、實價登錄之資料探勘等等，由於同時間還要攝影所以我並沒辦法記住全部的內容，加上黑客松時間並不算長，有些仍然在草創的階段，讓我們期待這些專案的有趣發展。也有些並不是在黑客松當天發生的，比方說新手村遊戲(下圖)及網站的重新設計，或者一直有在進行的專案如萌典、鄉民關心您及立院組，有些雖然並非直接與改造社會相關，但如同前面所描述的，當你沒辦法做大事時，期許自己是顆螺絲釘吧！
