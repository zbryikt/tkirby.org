新竹螢火蟲季 / 五指山螢海

隔了一週，這週二再度出發賞螢，這次拜訪的地方是新竹五指山附近的螢點。這帶距離北埔冷泉並不遠，也可以從竹東進入，但是平日上下班時間我從科學園區出發，並不想要與通勤車陣爭道，便從寶山經北埔進入。在計劃路線時，前陣子北埔冷泉道路坍坊的新聞著實讓我猶豫了一下，上公路局網站也得不到相關的資訊，不過前天看到有人分享道路已搶通、目前是單線通車的情形，就決定還是由北埔前往這個地方賞螢。

原本並沒有特別計畫在這一天要來拍；之前曾到彰化幫一位高中同學拍婚禮，前一天跟他約好要交付成品，結果在公司待太晚放了人家鴿子，想想不然隔天就約來一起拍螢火蟲吧，這樣保證不會放人家鴿子，於是地點查一查就約了，所以這一次我是與一位高中同學一同前往。他一路上跟我講說沒有這樣在半夜跑到山裡過啦、兩個人夜裡的山中讓人心頭小鹿亂撞什麼的，事實上拍攝的環境還不錯，只是真的只有我們兩個人在現場。偌大的山區，一戶住家，兩個遊客，滿滿的螢火蟲，賞螢品質 MAX 。

雖然之前去過北埔冷泉，但五指山區我還是第一次去，五指山正好在五峰、北埔、竹東三者的交界處，不過這裡螢火蟲似乎遍佈各處，頗難界定賞螢的點到底屬於哪個鄉鎮。過冷泉的路上也幾乎沒有路燈及住家，我想多少就連隨便的路邊都多少可以看到一些螢火蟲吧。

從科學園區大約花了四五十分鐘的車程開到目的地，心頭小鹿亂撞的同行友人忽然說想上廁所，身為都市人(?)的他又不好意思隨地來，恰好有間廁所就跑進去了。沒想到這裡連廁所都有螢火蟲。不曉得各位有邊上廁所邊看螢火蟲的經驗嗎？在廁所外的我往旁邊一看，整片的螢火蟲，從停車的地方到山腰邊、沿著山邊的道路接到產業道路上、山腰上，到處都有螢火蟲飛來飛去，數量大概是今年看到最多最廣的。



不過雖然前來的路上沒有路燈，這裡很矛盾的是有數盞路燈在周邊，光害頗嚴重，整個環境打得很亮，還分兩種顏色，一種是黃光另一種則是青光，所以雖然天氣多雲且沒有月光，曝光時間還是拉不長，構圖也受到限制，當下覺得很可惜，不過後來發現因此反而拍出了很有趣的照片。

路邊剛好有一灘小水塘，嘗試了一下拍攝螢火蟲倒影，難度頗高。可以看見水塘中反射出黃色的背景，那就是上面這盞黃色路燈所照亮的：


下圖中右有一隻螢火蟲在空中繞了一圈，倒影中也呈現出其影像，不過也許構圖再近一點會更好。主要的問題是螢火蟲來來去去，同時倒影要出來必須要有適當的角度，所以要有些耐心才比較有機會同時拍到其倒影。當然數量多的話這比較不會是問題，只是因為遠方有光害，曝光時間無法拉長，當構圖限制在水塘邊時同時拍到的數量仍然有限。


照片中的草叢裡發光的地方並不是螢火蟲的螢光，而是被遠方路燈所打亮的；由於有兩種色溫的燈，可以在下圖中看到有兩種不同的光。現場拍攝時覺得因為反差過大所以很難操作，但回家到電腦上看發現其實還好，反而讓場景多了一些變化。尤其高色溫的燈打出青綠的草，真的很像有螢火蟲在其中閃爍，我覺得比起一般暗處長曝有另一種不同的感覺。


在這裡拍攝到快八點，見螢量開始減少準備要回去了，在收拾之前我四處晃了一下，發現在道路右方的樹林裡也有一整片的螢火蟲；抬頭一看，往山上的步道也有許多螢火蟲；我再往反方向走，穿過一段鋪了石板的草皮，又見到一整片滿滿的螢火蟲。這裡螢火蟲遍布的範圍相當的廣，而且造訪當天螢火蟲的數量十分的多。

只是仍然很矛盾的是我穿過石板步道所到的這塊空地光害也是非常嚴重，幾乎整遍空地都籠罩在燈光之下，但螢火蟲卻相當的多。我覺得很有趣，一般都說螢火蟲有負趨光性，路燈會影響他們的生態；但這幾盞燈看起來設立在這邊應該也不是一兩天了，螢火蟲好像也不避諱的大喇喇的曝曬在燈光下求偶。也許不開燈螢火蟲會更多吧，但我只是路人，當地的事還是由當地人自己決定吧，所以我只好來拍個逆光人像螢光了：


除了利用刺眼的路燈逆光拍攝，因為樹木而產生的陰影替這個場景造出了非常強烈的反差。青橙色塊交錯著，亮處帶有清楚的景、而暗處則有美麗的螢光飛舞，在大光圈鏡頭的洗禮後，成品帶來的效果有別於以往所拍攝出來的印象。




這天拍到快九點，除了偶而路過的車輛以外，就只有我跟高中同學兩人，也許是平日人比較少吧，但事實上這一帶的民宿在螢火蟲季似乎房客並不少，有些民宿聽說在窗邊就可以看到螢火蟲了，所以也許明年可以安排個兩天一日遊：

D1 0800 台北 → 0930 峨眉十二寮湖畔咖啡(午餐) → 1300 北埔老街/二寮神木 → 1700 民宿(晚餐) → 賞螢 → 就寢
D2 0700 ~ 1100 五指山 → 1200 竹東大山背(午餐) → 1300 ~ 1600 騎龍古道/樂善堂 → 1900 台北

哇！美好的行程！讓我想起了以前揪團爬山的回憶啊！只好現在開始接受團報預約了！

這是今年第六個賞螢點，我想就以這裡做為今年螢火蟲巡禮的最後一站吧。雖然還有很多地點等著我去探尋，但是畢竟時間也不是那麼的多，還有其它的事要忙，再說慢慢的來拜訪人生會比較有趣味。到過了許多地方，每個其實都有其特色，我想將來若有機會，我會希望不只是來賞螢，也能同時了解當地的人文特色或是山林景觀，相信新竹是個美麗的地方，值得探索。
