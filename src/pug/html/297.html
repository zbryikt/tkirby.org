鏡頭的星芒測試

最近看著防潮箱裡的鏡頭，臨時起意想要看看每顆鏡頭的星芒各是長成什麼樣。

先來點基礎概念：

  • 星芒是光線穿過光圈時因繞射而產生的，入射光波長越長、鏡頭光圈越小則繞射越明顯。
  • 光芒數則與光圈葉片相關(更精確的說是與葉片間夾出的角相關)。
  • 一般來說奇數葉片產生兩倍葉片數的光芒，偶數則產生與葉片數同數量的星芒。
  • 圓形光圈產生的繞射光芒比較不明顯。
  • 幾何邊長不對稱的光圈形狀會造成比較不規則的星芒。
  • 在同樣的光圈幾何形狀下，長焦段的鏡頭一般來說會有比較優的星芒。
  • 拍攝時簡單的記法即是，光圈定胖瘦，快門定長短。

一般的鏡頭在縮小光圈後拍攝點光源時多半會出現星芒的現象，因此我認為星芒的型態應該也可以視為鏡頭的一個屬性或特色。之前曾經針對其中一部份的鏡頭做過測試，可以參考這個網頁(大屯山、金瓜石、茶壺山夜拍)。這次則是使用其它的鏡頭來拍攝星芒，並將結果列出於此。(先前的結果也一併列於此)

Nikon PC-E 24mm f/3.5D ED

一顆新入手的特別鏡頭，廣角移軸鏡，其星芒如同其它定焦鏡一樣優異。光圈最小可達 f/32，這張照片曝光 5 分鐘、光圈 f/19、 ISO 250。值得注意的是在照片左方有可見的鬼影出現。

 

Sigma 8mm f/3.5 EX DG Circular Fisheye

魚眼鏡頭無法安裝保護鏡，所以拍的時候不擦的話就會看到一顆一顆塵埃。基本上是六芒發散狀，有趣的是不同光圈時有不同的效果，請注意下面兩張圖的光圈)

8mm (8s, ISO 200, f/6.3)

8mm (30s, ISO 200, f/16)

Sigma 10-20mm f/4-5.6 EX DC HSM

同樣是六芒發散式的星芒

Nikon AF-S 14-24mm f/2.8G ED

Nikon 大三元廣角鏡，因為無法裝保護鏡，拍照時也沒有特別擦拭鏡頭，所以拍得一粒一粒的，不過不影響觀察星芒就是了。可以由遠方星芒看出不規則的收斂式 18 芒星，除了長度不平均外，放大後可以看到每一道星芒其實明顯是由數道組成的。

16mm (8s, ISO500, f/22)

24mm (8s, ISO 500, f/22)

Sigma 24-70mm f/2.8 IF EX DG HSM

18芒發散式，類似 Nikon 14-24mm，遠看也是不錯。

24mm (8s, ISO 320, f/16)

70mm (8s, ISO 320, f/16)，局部裁切。

Nikon AF-S 24mm f/1.4G ED

廣角大光圈定焦鏡，18 芒收斂式，光芒以一長一短的規率呈現。

24mm (4s, ISO 200, f/16)，這張拍的時候保護鏡忘了拿下來。當時正在下雨，所以鏡片上有水滴。

24mm (8s, IS0 320, f/16)，順著風向拍並且拿下保護鏡，所以就沒有暈開的水滴了。這張是局部裁切。

Sigma 30mm f/1.4 EX DC HSM

拍人很不錯的鏡頭，最近對焦距離為 45cm 稍微有點長。八芒收斂式，很厚實很對稱，很容易拍出星芒。可惜是 APSC 專用鏡，在APSC 機型上 30mm 又不夠廣，若要拍夜景星芒可能會有點不足。

30mm (2s, ISO 640, f/13)，隨便拍星芒就很長。(局部裁切)

Nikon AF 35mm f/2D

很小巧的一顆鏡頭，發出14 芒收斂式星芒，同樣的很容易拍出厚實對稱的星芒，同樣也有光芒一長一短的現象。

35mm (10s, ISO 400, f/22)

35mm (10s, ISO 400, f/22) 局部裁圖

Sigma 50-500mm f/4.5-5.6 DG OS HSM

18芒星發散式，50mm 及 500mm 端都有拍攝，不過 500mm 沒有適當的景色所以拍不出效果。好像還不錯看。

50mm (13s, ISO 320, f/22)

500mm (13s, ISO 320, f/22) 風很大，腳架不夠穩，所以蠻晃的，試了幾張就先放棄了。

Sigma 50mm f/1.4 EX DG HSM

Sigma 的標準大光圈，表現普通，18 芒星收斂式但有點開叉，間距也不是相當整齊。

50mm (6s, ISO 400, f/16)

50mm (5s, ISO400, f/16) 局部裁圖

Nikon AF 50mm f/1.8D

超值標準大光圈人像鏡，14芒收斂式星芒也很漂亮，而且不太需要縮光圈就能做得出來，不過似乎某些狀況下會出現光源的鬼影？

Nikon AF-S 85mm f/1.4G

18芒收斂式星芒，細細長長的星芒相當好看。必要的話可能得縮一下快門以減短星芒長度。

85mm (8s, ISO 400, f/16)

Nikon AF-S VR Micro 105mm f/2.8G ED? 微距鏡

也很適合拍人的百微鏡，18芒收斂式星芒，星芒感覺有點虛胖，放大來看會發現很多道光芒其實都是由一條以上組成的。感覺不會是拍攝星芒的首選。

105mm (13s, ISO 500, f/32)

局部放大

Lensbaby Composer

這算是顆拍攝特殊效果的鏡頭，有點像玩具，價錢也不高，光圈要手動更換，而且是圓的。他有提供創意光圈組 (簡單的說就是沒開洞的光圈片，自己動手開光圈)，所以也許可以自行開出模擬 N 片葉片的光圈。我則是直接使用現成的 f/22 光圈。

50mm (8s, ISO 400, f/22) 由於圓形光圈的關係，星芒不明顯，加上 Lensbaby Composer 畫質本身就不好，拍出的效果不會讓人太滿意

總結

由於拍攝當天刮風下雨，沒有拍相當多的照片，不過就表現鏡頭星芒的特性來說，應該是足夠的了。不過這次並沒有針對不同光圈下的星芒變化一併做測試而只測試大部份鏡頭在光圈最小時星芒的樣子而已。
